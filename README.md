這是一個為 **祥安生命** 打造的專業禮儀服務網站，專注於提供透明的喪葬費用資訊與 24 小時溫馨關懷。

## 🔗 線上預覽
[👉 點此進入祥安生命官網](https://shiangan.github.io/Shiangan/)

## 🛠 核心技術與特色
- **專業排版**：使用自訂字體（uwen, Iansui）與響應式設計 (RWD)。
- **自動化標題縮放**：運用 CSS `clamp()` 與 `fit-text-line` 確保長標題在各裝置完美呈現。
- **動態環境效果**：內建流星背景動畫與平滑的手風琴 FAQ 互動。
- **SEO 結構化資料**：整合 JSON-LD 標記，強化搜尋引擎對生命禮儀服務的索引。

## 📂 資料夾結構
- `/css`: 包含模組化後的 `style.css`。
- `/js`: 包含 `script.js` (處理選單、滾動與手風琴)。
- `/fonts`: 品牌專用字體檔。
- `/images`: 網站圖案與 Logo。
